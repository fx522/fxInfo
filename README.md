# 如何使用
* 建立数据库（根目录fxinfo_db.sql文件）
* 配置站点数据库信息
  位于application/config/database.php文件中（填入数据库用户名密码和数据库名）
* 将应用部署到Web容器中运行即可.
* 初始用户名和密码 都为： fx-admin1204
